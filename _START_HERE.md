## üìù Converting to PDF

### Method 1: Using Pandoc (Recommended)

```bash
# Install Pandoc first: https://pandoc.org/installing.html

# Convert cover page + report to single PDF
pandoc COVER_PAGE.md PROJECT_REPORT.md -o Daksh_Project_Report.pdf --toc

# Or convert individually
pandoc COVER_PAGE.md -o Cover_Page.pdf
pandoc PROJECT_REPORT.md -o Project_Report.pdf --toc
pandoc PROJECT_SUMMARY.md -o Project_Summary.pdf
```

### Method 2: Using VS Code

1. Install "Markdown PDF" extension
2. Open any .md file
3. Right-click ‚Üí "Markdown PDF: Export (pdf)"
